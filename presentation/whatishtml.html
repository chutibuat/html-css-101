HTML คืออะไร?

HTML คือ ภาษาหลักที่ใช้ในการเขียนเว็บเพจ โดยใช้ Tag ในการกำหนดการแสดงผล HTML ย่อมาจากคำว่า Hypertext Markup Language โดย Hypertext หมายถึง ข้อความที่เชื่อมต่อกันผ่านลิ้ง (Hyperlink) Markup language หมายถึงภาษาที่ใช้ Tag ในการกำหนดการแสดงผลสิ่งต่างๆที่แสดงอยู่บนเว็บเพจ

HTML Basic Examples

HTML Documents

เอกสาร HTML ทั้งหมดจะต้องเริ่มต้นด้วยการประกาศประเภทเอกสาร: <!DOCTYPE html>.

เอกสาร HTML นั้นเริ่มต้นด้วย และปิดท้ายด้วย </html>.

ส่วนที่มองเห็นได้ของเอกสาร HTML อยู่ระหว่าง <body> และ </body>.

<!DOCTYPE html>
<html>
<body >

<h1>My First Heading</h1 >
<p>My first paragraph.</p >

</body >
</html>

HTML Links

ลิงค์ HTML ถูกกำหนดด้วย Tag <a>

<a href="https://www.lipsum.com/" target="blank" > This is a link</a>

HTML Images

รูปภาพถูกกำหนดด้วย Tag <img>

โดยประกอบไปด้วย attributes source file (src), ข้อความอธิบายรูปภาพ (alt), width, height

<img src="https://api.adorable.io/avatars/285/2.png" alt="รูปภาพ Avatar" width="104" height="142" >

HTML Buttons

ปุ่มถูกกำหนดด้วย Tag <button>

<button>Click me</button >

HTML <div> Tag

div คือ แท็กประเภทนึงของ html ใช้สำหรับครอบวัตถุที่เราต้องการ เพื่อจัดรูปแบบต่างๆ ให้กับวัตถุในตำแหน่งนั้นๆ สามารถใส่ค่าไปตรงๆ ใน div นั้น หรือ ใส่ id กับ class เพื่อใช้อ้างอิงกับการจัด CSS ได้ตามความเหมาะสม โดยใช้ Tag <div>

<div class ="myDiv" >
  <h2 > This is a heading in a div element </h2 >
  <p > This is some text in a div element. </p >
</div >
